Expense Tracker

In SE 1860, I worked together in a group to create a simple web-based expense tracker that allows me to add, view, filter, and delete personal expenses. I created the interface that lets the user enter an expense name, amount, and category, then automatically displays each expense with its details and updates the total amount spent. I used JavaScript and HTML to handle expense management, filtering, and real-time updates. The layout is styled with embedded CSS to keep the design clean, responsive, and easy to use. It is visible on my GitHub profile.
